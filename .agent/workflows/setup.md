# 開発環境セットアップと起動

このワークフローは、プロジェクトの開発環境を初期化し、開発サーバーを起動するために使用します。コマンドの呼び出しで「/setup」等として利用できます。

## ステップ

// turbo-all
### ステップ1: 依存関係のインストール
- `npm install` を実行してプロジェクトの依存関係をインストールします。
- 実行ディレクトリ: `.`
- SafeToAutoRun: true

### ステップ2: 型チェックとLinterの実行
- `npm run lint` を実行してコードに静的なエラーがないか確認します。
- 実行ディレクトリ: `.`
- SafeToAutoRun: true

### ステップ3: テストの実行 (ドライラン)
- `npm run test` を実行して現状のテスト通過状況を確認します。
- 実行ディレクトリ: `.`
- SafeToAutoRun: true

### ステップ4: 開発サーバーの起動
- `npm run dev` を実行して Vite 開発サーバーを起動します。
- 実行後、ブラウザでアプリケーションにアクセスできることを確認してください。
- 実行ディレクトリ: `.`
- SafeToAutoRun: true
