## トラブルシューティング

### プラン作成ボタンが無反応
→ DevTools Console でエラーを確認。Supabase への INSERT クエリが失敗していないか確認。

### 「権限がありません」エラー
→ RLS (Row Level Security) ポリシーの問題。Supabase → Authentication → Policies → plans テーブルのポリシーを確認。

### 価格が保存されない
→ 入力値のバリデーションを確認。数値のみ許可、小数点の扱い等。
