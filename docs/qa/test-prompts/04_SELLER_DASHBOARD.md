## TC-04-01: ダッシュボードの基本表示

**手順**:

**期待結果**:
- [ ] 見出し「ダッシュボード」が表示される

## TC-04-02: KPIカード6枚の表示と値

**手順**:
1. ダッシュボードのKPIカードを確認

**期待結果**:
- [ ] 「有効会員数」カード — 数値が表示される、クリックで `/seller/members` に遷移
- [ ] 「アクティブプラン」カード — 数値が表示される、クリックで `/seller/plans` に遷移
- [ ] 「月間売上」カード — 金額がフォーマットされて表示される（¥形式）
- [ ] 「解約率」カード — パーセンテージが表示される
- [ ] 「今月の新規」カード — 数値が表示される
- [ ] 「Webhook数(今日)」カード — 数値が表示される
- [ ] すべてのカードにアイコンが表示される

---

**期待結果**:
- [ ] `/seller/members` に遷移する

**手順**:
2. 戻って「アクティブプラン」カードをクリック

**期待結果**:
- [ ] `/seller/plans` に遷移する

## TC-04-04: Stripe Connect ステータス表示

**手順**:
1. ダッシュボードの「Stripe Connect」セクションを確認

**期待結果**:
- [ ] 「Stripe Connect」ラベルが表示される
- [ ] 「決済受付状態」の表記
- [ ] ステータスバッジが表示される（「有効」「未開始」「審査中」「制限あり」のいずれか）
- [ ] バッジの色がステータスに応じて適切に変わる

---

## TC-04-05: Discord Bot 未接続警告

**手順**:
1. Discord Botが未接続の状態でダッシュボードを確認

**期待結果**:
- [ ] 警告バナー「Discord Bot未接続」が表示される
- [ ] 説明「ロールの自動付与が行われません。Discord設定を確認してください。」
- [ ] 「Discord設定へ」ボタンが表示される

**手順**:
2. 「Discord設定へ」をクリック

**期待結果**:
- [ ] `/seller/settings/discord` に遷移する

---

## TC-04-06: お知らせバナーの表示

**手順**:
1. ダッシュボードのお知らせセクションを確認

**期待結果**:
