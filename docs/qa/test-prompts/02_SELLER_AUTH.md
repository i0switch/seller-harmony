# テスト02: Seller 認証（新規登録・ログイン）

> **カテゴリ**: 認証  
> **優先度**: P0 (Critical)  
> **推定所要時間**: 15分  
> **前提条件**: 未認証状態で開始

---

## AIエージェントへの指示

```
以下のテストケースでSeller（販売者）の新規登録とログインフローを検証してください。
テスト環境: https://preview--member-bridge-flow.lovable.app/
Supabase Auth経由の認証です。Mock APIモードの場合はモック認証が動作します。
バリデーションエラー、エラーメッセージの表示を厳密に確認してください。
```

---

## TC-02-01: 新規登録ページの表示

**手順**:
1. `/seller/signup` にアクセス

**期待結果**:
- [ ] 見出し「🎤 販売者登録」が表示される
- [ ] サブタイトル「ファンクラブを始めましょう」が表示される
- [ ] 以下のフォームフィールドが表示される:
  - 「表示名」入力欄（placeholder: 「クリエイター名」）
  - 「メールアドレス」入力欄（placeholder: 「you@example.com」）
  - 「パスワード」入力欄（placeholder: 「8文字以上」）
- [ ] 「利用規約およびプライバシーポリシーに同意します」チェックボックスが表示される
- [ ] 「アカウント作成」ボタンが表示される
- [ ] 「既にアカウントをお持ちですか？ ログイン」リンクが表示される

---

## TC-02-02: 新規登録バリデーション — 空送信

**手順**:
1. `/seller/signup` にアクセス
2. 何も入力せず「アカウント作成」ボタンをクリック

**期待結果**:
- [ ] 「表示名を入力してください」エラーメッセージが表示される
- [ ] 「有効なメールアドレスを入力してください」エラーメッセージが表示される
- [ ] 「パスワードは8文字以上にしてください」エラーメッセージが表示される
- [ ] フォームは送信されない

---

## TC-02-03: 新規登録バリデーション — 個別フィールド

**手順**:
1. 表示名に「テスト販売者」を入力
2. メールアドレスに「invalid-email」（不正な形式）を入力
3. パスワードに「1234」（8文字未満）を入力
4. 利用規約チェックボックスを**チェックしない**
5. 「アカウント作成」をクリック

**期待結果**:
- [ ] 表示名のエラーは表示されない
- [ ] 「有効なメールアドレスを入力してください」が表示される
- [ ] 「パスワードは8文字以上にしてください」が表示される
- [ ] 「利用規約に同意してください」が表示される

---

## TC-02-04: 新規登録バリデーション — 正しい入力

**手順**:
1. 表示名: 「テスト販売者」
2. メールアドレス: 「test-seller@example.com」
3. パスワード: 「password123」
4. 利用規約: チェックする
5. 「アカウント作成」をクリック

**期待結果**:
- [ ] バリデーションエラーが表示されない
- [ ] ボタンがローディング状態になる（スピナーまたはテキスト変更）
- [ ] 成功時: `/seller/onboarding/profile` にリダイレクトされる
- [ ] エラー時（メール重複等）: エラーメッセージが赤帯で表示される

---

## TC-02-05: 新規登録→ログインページへの遷移

**手順**:
1. `/seller/signup` にアクセス
2. 「既にアカウントをお持ちですか？ ログイン」リンクをクリック

**期待結果**:
- [ ] `/seller/login` に遷移する
- [ ] ログインフォームが正しく表示される

## TC-02-07: ログインバリデーション — 空送信

2. 何も入力せず「ログイン」をクリック

**期待結果**:
- [ ] 「メールアドレスとパスワードを入力してください」エラーが表示される
- [ ] フォームは送信されない

---

## TC-02-08: ログインバリデーション — 不正な認証情報

**手順**:
1. メールアドレス: 「wrong@example.com」
2. パスワード: 「wrongpassword」
3. 「ログイン」をクリック

**期待結果**:
- [ ] 「ログインに失敗しました。メールアドレスとパスワードを確認してください。」エラーが表示される

---

## TC-02-09: ログイン成功

**手順**:
1. 有効なSellerアカウントでログイン
   - メール: Supabaseに登録済みのテストアカウント
   - パスワード: テストアカウントのパスワード
2. 「ログイン」をクリック

**期待結果**:
- [ ] ボタンがローディング状態になる
- [ ] ログイン成功後、以下のいずれかにリダイレクト:
  - オンボーディング未完了の場合: `/seller/onboarding/profile`（または該当ステップ）
  - オンボーディング完了済みの場合: `/seller/dashboard`
- [ ] エラーが表示されない

---

## TC-02-10: ログイン→新規登録ページへの遷移

**手順**:
1. `/seller/login` にアクセス
2. 「初めてですか？ 新規登録」リンクをクリック

**期待結果**:
- [ ] `/seller/signup` に遷移する
- [ ] 新規登録フォームが正しく表示される

---

## TC-02-11: パスワード入力のセキュリティ

- [ ] 同様にマスク表示
