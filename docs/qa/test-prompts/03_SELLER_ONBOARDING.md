## TC-03-12: ステップ2 — Discord検証の実行

**手順**:
1. サーバーID: 「1234567890123456789」を入力
2. ロールID: 「9876543210987654321」を入力
3. 「Discord設定を検証」をクリック

**期待結果**:
- [ ] ボタンが「検証中...」に変わる
- [ ] 検証完了後、結果が表示される:
  - 成功時: バッジ「検証OK」＋チェック項目がすべて緑:
    - 「Bot導入済み」✓
    - 「Manage Roles権限あり」✓
    - 「対象ロール存在」✓
    - 「Bot役職が対象ロールより上位」✓
  - 失敗時: バッジ「検証NG」＋エラーコードと日本語説明

---

## TC-03-13: ステップ2 — Discord検証エラーのエラーコード表示

**手順**:
1. 検証NGの場合、以下のエラーコードと対応メッセージを確認

**期待結果**:
- [ ] エラーコードが表示される場合、以下のいずれか:
  - `DISCORD_GUILD_ACCESS_DENIED` — Botがサーバーにアクセスできない
  - `DISCORD_MANAGE_ROLES_MISSING` — ロール管理権限がない
  - `DISCORD_ROLE_NOT_FOUND` — 指定ロールが見つからない
  - `DISCORD_ROLE_HIERARCHY_INVALID` — Bot役職が対象ロールより下位
- [ ] 各エラーに対応する日本語の説明テキストが表示される
