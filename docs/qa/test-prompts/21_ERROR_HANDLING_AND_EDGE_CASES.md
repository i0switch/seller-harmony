## TC-21-04: 空文字列・特殊文字の入力

**手順**:
1. `/seller/plans/new` のプラン名に以下を入力してテスト:
   - スペースのみ: 「   」
   - 特殊文字: 「<script>alert('xss')</script>」
   - 絵文字: 「🎤テスト🎵」
   - 超長文: 1000文字以上の文字列
2. 検索バーに以下を入力:
   - 特殊文字: 「%&'\"<>」
   - SQL injection風: 「'; DROP TABLE members;--」

**期待結果**:
- [ ] スペースのみ: バリデーションエラー、またはトリムされる
- [ ] XSS: HTMLがエスケープされ、スクリプトが実行されない
- [ ] 絵文字: 正常に保存・表示される
- [ ] 超長文: UIが崩れない（テキストが折り返しまたは切り詰め）
- [ ] SQL injection: アプリがクラッシュせず、検索結果が空になる
